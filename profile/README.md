## 🌿 About

**Vertex Kashmir Holidays** is a travel and tourism company dedicated to providing exceptional travel experiences across Jammu & Kashmir.

Our platform enables travelers to discover destinations, explore curated tour packages, generate personalized itineraries, and book memorable journeys — all in one place. We combine deep local knowledge with modern technology to make Kashmir travel accessible, reliable, and unforgettable.

## 🛠️ Technology

We're building our platform entirely in-house with a modern, scalable stack.

```text
Frontend
├── Next.js 15 (App Router)
├── React 19
├── TypeScript 5
└── Tailwind CSS + Framer Motion

Backend
├── Next.js API Routes
├── Prisma ORM
└── Neon Serverless Postgres

Authentication
└── Auth.js v5

Payments
└── Razorpay

Communication
└── Nodemailer (Booking confirmations & OTPs)

Deployment
└── Vercel (Global Edge Network)
```
